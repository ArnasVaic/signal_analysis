# Recursive diagram

## Task description

Rekurentinės diagramos. Slenkstinio atstumo parametrą $r$ parinkite pagal vartotojo įvestą pageidaujamą juodų taškų (pikselių) rekurentinėje diagramoje procentą. Pademonstruokite algoritmo veikimą, pateikite rezultatų interpretavimo pavyzdžių įvairiais atvejais. Palyginkite gaunamus rezultatus skaičiuodami su
įvairiais algoritmo parametrais $D$ ir $d$, bei kai naudojamos skirtingos metrikos: $L_2$ (Euklidinė); $L_1$ (Manheteno); $L_{\infty}$ (maksimumo). Referate išsamią (sudarytą iš visų paminėtų tyrimų) analizę pateikite tik vienam pasirinktam (kuriam Jūsų nuomone gaunami vaizdingiausi rezultatai) signalui, kitiems tirtiems signalams referate galima pateikti ne visas tyrimus iliustruojančias iliustracijas (tačiau atsiskaitymo metu
gali buti paprašyta pademonstruoti ir praleistų tyrimų veikimą kompiuterio ekrane).
